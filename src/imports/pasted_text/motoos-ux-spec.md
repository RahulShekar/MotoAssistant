# Minimal Navigation UX (Critical)

While riding, the rider should only see 5 action buttons.

### Bottom Navigation

🏠 Home

🗺 Navigate

🧠 Intelligence

✅ Readiness

👤 Profile

---

### Floating Button

🎤 Ask Moto

This is the only prominent action button.

Everything else should be voice-first.

Examples:

> "Moto, where is the next fuel station?"

> "Moto, any rain ahead?"

> "Moto, nearest mechanic?"

> "Moto, how much distance remaining?"

The goal is that the rider rarely touches the screen while riding.

# MotoOS Core Promise

**"Ride. MotoOS watches the road ahead."**

Every screen, alert, and AI response should reinforce that single promise. That is what makes MotoOS different from generic navigation apps.
